Sobre Nosotros

Esto es solo una demo de un Blog hecho con Node.js mediante el framework VUE.js para el frontend, utilizando como base de datos MongoDB.

El proyecto permite listar los ultimos 5 articulos creados en la pagina principal. Todos los articulos que hay en la base de datos aparecerán en la página de Blog.

Tambien dispone de un formulario de contacto en la pagina Contacto. El formulario tiene un validador, aunque no tiene implementada la funcionalidad de enviar emails.

El Sidebar contiene 2 botones totalmente funcionales. Uno para crear un nuevo articulo y un buscador para buscar articulos por palabras. Al abrir un articulo existente, se han implementado botones para modificar y eliminar articulos.

Como posible mejora para un update futuro, seria implementar un login, para que solo personas logeadas puedan crear,modificar y eliminar artículos, mientras el resto puedan leer los artículos.
